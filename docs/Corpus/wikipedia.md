# 维基百科语料库

[语料库](https://zh.wikipedia.org/wiki/%E8%AF%AD%E6%96%99%E5%BA%93)一词在语言学上意指大量的文本，通常经过整理，具有既定格式与标记；事实上，语料库英文 "text corpus" 的涵意即为 "body of text"。
